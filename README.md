# Student Management System

A command-line Student Management System built with Python and SQLite that demonstrates object-oriented programming, database design, and CRUD operations. This project is being developed as a learning project to strengthen my backend development skills while following clean, modular software design principles.

## Features

- Add new students to a SQLite database
- View all students
- Search for students by ID
- Update student GPA
- Delete students
- Prevent duplicate student IDs
- Interactive command-line menu
- Object-oriented design using a `Student` class
- Modular project architecture

## Technologies Used

- Python 3
- SQLite3
- SQL
- Git & GitHub

## Project Structure

```
Student-Management-System/
│
├── main.py          # Program entry point
├── menu.py          # Command-line user interface
├── database.py      # Database operations and SQL queries
├── student.py       # Student class
├── utils.py         # Helper functions (future use)
├── students.db      # SQLite database (ignored by Git)
└── README.md
```
