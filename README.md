# Student Management System

A Python-based Student Management System that demonstrates object-oriented programming, SQLite database integration, and SQL operations. This project is being developed as a learning project to strengthen my backend development skills and understanding of databases.

## Features

- Add new students to a SQLite database
- Prevent duplicate student IDs
- Retrieve all students from the database
- Search for a student by ID
- Object-oriented design using a `Student` class
- Modular project structure

## Technologies Used

- Python 3
- SQLite
- SQL
- Git & GitHub

## Project Structure

```
Student-Management-System/
│
├── database.py      # Database functions and SQL queries
├── student.py       # Student class

├── menu.py          # (Planned) Interactive menu
├── utils.py         # (Planned) Helper functions
├── students.db      # SQLite database (ignored by Git)

```
